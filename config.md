# Configuration

This addon has three settings:

- `show_tooltip`
  - `true`: show small confirmation tooltips after automatic cleanup and purge.
  - `false`: run silently.

- `auto_run_startup`
  - `true`: run a collection-wide `&nbsp;` purge when Anki starts.
  - `false`: only purge manually from the Tools menu.

- `purge_shortcut`
  - Keyboard shortcut for `Tools -> nbsp Remover -> Purge collection from &nbsp;`.
  - Default: `Ctrl+Alt+P`.

## Where to change settings

Open `Tools -> nbsp Remover -> Options`, then click `Save`.
